عبور از تحریم های گریدل

این سرویس برای دانلود دیپندنسی های گریدل بدون محدودیت و عبور از تحریم ها طراحی شده است.
شما می‌توانید با تغییر تنظیمات پروژه خود، به‌سادگی از این Mirror استفاده کنید.

نحوه استفاده در Gradle (Kotlin DSL)

تیکه کد maven("https://en-mirror.ir") را به لیست ریپازیتوری های خود در فایل settings.gradle.kts اضافه کنید.

pluginManagement { repositories { maven("https://en-mirror.ir") } } dependencyResolutionManagement { repositories { maven("https://en-mirror.ir") } }

محدود کردن Mirror به یک Repository خاص

به‌طور پیش‌فرض، همه مخازن (Google، Maven Central، JitPack) پشتیبانی می‌شوند. اگر می‌خواهید فقط یک مخزن خاص را Mirror کنید، از مسیر مخصوص استفاده کنید:

maven("https://en-mirror.ir/jitpack")

این Mirror شامل مخازن Google، Maven Central و JitPack است.
